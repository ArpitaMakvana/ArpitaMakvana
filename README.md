## 💫 About Me

I am a **Machine Learning Developer** passionate about building intelligent systems, exploring Deep Learning architectures, and developing interactive web applications. I love bridging the gap between advanced AI modeling (especially in medical diagnostics and cognitive load analysis) and modern user interfaces.

*   🏫 **Education:** Studying Computer Engineering at **Sardar Vallabhbhai National Institute of Technology (SVNIT), Surat**.
*   💻 **Currently Building:** Temporal Deep Learning models to analyze cognitive patterns, alongside developing responsive web applications.
*   🔍 **Focus Areas:** Deep Learning, Medical Image Classification (Colon Cancer Detection), Cognitive Load Modeling, and Full-Stack Web Development.
*   ✍️ **Interests:** Tech Blogging, Public Speaking, UI/UX Design, and Prompt Engineering.

## 🚀 Featured Projects

### 🧠 [C.O.G.N.I.T](https://github.com/ArpitaMakvana/C.O.G.N.I.T)
*Modeling cognitive load as an emergent temporal phenomenon from micro-interaction dynamics in Deep Learning.*
*   **Technologies:** HTML, JavaScript, Deep Learning concepts (temporal sequencing)
*   **Core Highlights:**
    *   Designed a model framework to evaluate real-time user cognitive load states using micro-interaction indicators.
    *   Models dynamic behavioral patterns to dynamically optimize interface responsiveness.

### 🩺 [MED_COLON_CANCER](https://github.com/ArpitaMakvana/MED_COLON_CANCER)
*Deep learning-based classification and analysis of colon cancer histopathological images.*
*   **Technologies:** Python, TensorFlow/PyTorch, Computer Vision, Jupyter Notebooks
*   **Core Highlights:**
    *   Built an end-to-end medical classification pipeline utilizing histopathology images.
    *   Achieved high-accuracy diagnostic predictions by applying advanced Convolutional Neural Network (CNN) topologies.

### 🥗 [Healthy Food](https://github.com/ArpitaMakvana/Healthy_food)
*An interactive web application focused on meal planning, nutrition tracking, and healthy recipes.*
*   **Technologies:** JavaScript, HTML5, CSS3
*   **Core Highlights:**
    *   Developed a clean, responsive client-side UI for recipe filtering and nutrition analysis.
    *   Designed dynamic food search algorithms to optimize recipe discovery.

### 🌌 [SpaceWEB22](https://github.com/ArpitaMakvana/SpaceWEB22)
*An immersive astronomy and space-exploration themed web interface.*
*   **Technologies:** JavaScript, CSS3, HTML5
*   **Core Highlights:**
